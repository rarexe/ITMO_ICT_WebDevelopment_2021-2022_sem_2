    3.4 test-patch.py

    Написать 3 PATCH запроса

    Тесты создают экземпляры для каждой из трех моделей (Person, Climbing, Club). Далее при помощи patch запроса получают код выполнения операции, который должен быть равен 200 для подтверждения правильной работы изменения данных, а также для прохождения теста измененные данные должны совпадать с объектом data, который содержит в себе ту информацию, которая должна быть в экземпляре если изменения прошли успешно.
